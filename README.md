# Goes Hard Inc. Minecraft Server Preservation Project
Find the project here: http://goeshardinc.com/

## About
Welcome to what I've been wasting my time on at work for the past few weeks! It started out as a small scale project to show off Me and my girlfriend's forever world. And now it's expanded to this entire history of Minecraft servers. And led to me learning how to self-host a web server, and then learn how to self-host a much better web server.

## Contribution
Contributions would definitely be appreciated! Though there's a few things to know before starting.
1. The world downloads, modpack files, and BlueMap tiles have not been added to the github due to size limitations
2. Additionally, there's a specific configuration you need for nginx to get the BlueMap maps to display properly
3. All of that to say, it's unlikely that we'll be changing much BlueMap stuff moving forward, so it's unlikely that you'll need to worry about that
4. If you would like those larger files that aren't stored on the repo, reach out to me on discord and I can get them to you

If you'd like to contribute to this project, just request access, I'd love to see what else we can do with this.
